Protected isOriginAllowed

  • isOriginAllowed(origin: undefined | string, params: CORSFilterParams): boolean
    • Defined in web-service/src/helpers.ts:75

    Check if the given origin is allowed to make a CORS request.

    The CORS specification recommends a server to return WebStatus.FORBIDDEN if a CORS request is denied. You can do that by throwing a WebError instead of returning false, like this:

    protected isOriginAllowed(origin: string | undefined, params: CORSFilterParams): boolean {
    if (origin === 'https://example.com) {
        return true;
    }
    else {
        throw new WebError(WebStatus.FORBIDDEN, `CORS request from origin ${origin} denied`);
    }
}

    Parameters

    • origin: undefined | string

      The value of the origin header, or undefined if the header was not provided.

    • params: CORSFilterParams

      Request params.

    Returns boolean

    true if the request is allowed, else false.
